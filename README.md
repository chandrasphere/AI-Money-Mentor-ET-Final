1. Architecture Document 
System Design & Logic:

Frontend Layer: Built with React 19 and Tailwind CSS for a responsive, mobile-first experience.

AI Orchestration Layer: Powered by Gemini 3 Flash Preview to handle complex financial reasoning and tax laws.

Data Visualization: Uses Recharts to transform raw user data into an intuitive 6-dimension Money Health Score.

Error Handling: Built-in guardrails in the prompt engineering to ensure disclaimer-first financial advice.

2. Impact Model
Business & Social Impact:

Cost Reduction: Professional financial planning in India costs ₹25,000+ per year. Our AI Mentor provides it at Zero Cost, making it accessible to 100% of the population.

Time Efficiency: Manual portfolio analysis and tax planning take hours; our agent completes a full X-Ray in <10 seconds.

Financial Inclusion: Targets the 14 crore+ demat account holders who currently "fly blind" without a formal plan.

Tax Optimization: Identifies missing deductions and compares regimes instantly, potentially saving a middle-class user ₹20,000 - ₹50,000 annually in taxes.
